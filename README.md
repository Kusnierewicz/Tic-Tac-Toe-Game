is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. Please see [Wikipedia article](https://en.wikipedia.org/wiki/Tic-tac-toe) for extended info.

## Installation

Save Tic-Tac-Toe-Game folder in your chosen location

And then inside of it execute:

    $ bundle

## Usage

To play the game run:

    ruby Tic-Tac-Toe-Game\run_game\play.rb
